# Exercice

## Exo 01
Réaliser un composant React qui permet d'afficher si un nombre est : pair, impair ou égale à zéro. 

Un texte doit être afficher dans une balise paragraphe : 

- Si le nombre est pair : 
> Le nombre 42 est pair.

- Si le nombre est impair : 
> Le nombre 13 est impair.

- Si le nombre est 0 :
> Le nombre est zéro!
