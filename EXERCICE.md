# Exercice

## Exo 01
Réaliser un composant React qui permet d'afficher si un nombre est : pair, impair ou égale à zéro. 

Un texte doit être afficher dans une balise paragraphe : 

- Si le nombre est pair : 
> Le nombre 42 est pair.

- Si le nombre est impair : 
> Le nombre 13 est impair.

- Si le nombre est 0 :
> Le nombre est zéro!

## Exo Formulaires

![énoncé exercice formulaire](/public/exoform.png)

Dans un seul composant, créer un formulaire qui permet d'encoder :
* Le prix de la note totale d'un restaurant (requis + nombre décimal > 0)
* Le montant du pourboire en % (vous pouvez faire un select avec plusieurs valeurs prédéfinies, ce sera encore mieux que laisser l'utilisateur taper) (requis)
* Nombre de personnes à partager l'addition (requis, nombre entier > 0)

Quand on clique sur Valider, fait le calcul du
* montant du pourboire
* montant total (note + pourboire)
* montant par personne

Quand on clique sur le bouton Réinitialiser, remet tout à 0
